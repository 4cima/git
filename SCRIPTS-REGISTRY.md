### 1. sync-to-turso-ultra-fast.js
**المسار:** `scripts/sync-to-turso-ultra-fast.js`

**الوظيفة:** مزامنة الأفلام والمسلسلات إلى Turso بسرعة عالية

**الوصف:**
- يعمل على الأفلام والمسلسلات معاً
- يستخدم batch operations (100 عمل/دفعة)
- 10 دفعات متزامنة
- معالجة تصادم slug مع fallback splitting
- تسجيل الأخطاء في `turso-sync-failures.jsonl`

**الاستخدام:**
```bash
node scripts/sync-to-turso-ultra-fast.js
```

**السرعة:** ~1,000 عمل/دقيقة

**الحالة:** ✅ جارية (Process 7)

## 🚀 سكريبتات التحسين والمعالجة

### 13. generate-ai-overviews.js
**المسار:** `generate-ai-overviews.js`

**الوظيفة:** توليد وصف عربي للأعمال التي بدون overview_ar باستخدام الذكاء الاصطناعي

**الوصف:**
- يولد وصف عربي احترافي للأعمال التي لديها overview_en لكن بدون overview_ar
- يستخدم 6 مفاتيح AI: GROQ, OPENROUTER (3), XAI, MISTRAL
- معالجة ذكية بالدفعات (50 عمل/دفعة)
- إعادة محاولة تلقائية عند الفشل (3 محاولات)
- يعالج ~140,000 عمل

**الاستخدام:**
```bash
# استخدام GROQ (افتراضي - سريع ومجاني)
node generate-ai-overviews.js

# استخدام OpenRouter (3 مفاتيح = 3× سرعة)
AI_PROVIDER=OPENROUTER node generate-ai-overviews.js

# استخدام XAI (Grok)
AI_PROVIDER=XAI node generate-ai-overviews.js

# استخدام Mistral
AI_PROVIDER=MISTRAL node generate-ai-overviews.js
```

**السرعة:** 
- GROQ: ~25 عمل/ثانية
- OpenRouter (3 مفاتيح): ~75 عمل/ثانية
- XAI: ~20 عمل/ثانية
- Mistral: ~30 عمل/ثانية

**الحالة:** ✅ جاهز (لم يُشغَّل بعد)
