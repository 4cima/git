## 📊 البيانات المعروضة

### ✅ Stats Bar (في صف واحد أفقي):
1. **التقييم** (vote_average) - مثال: 8.2
2. **السنة** (release_date / first_air_date) - مثال: 2022
3. **المدة** (runtime) - للأفلام فقط - مثال: 2س 15د
4. **عدد التقييمات** (vote_count) - مثال: 3,330
5. **اللغة الأصلية** (original_language) - مثال: en
6. **النوع** (media_type) - فيلم أو مسلسل
7. **Type** (type) - للمسلسلات فقط - مثال: Scripted, Documentary

### ✅ Sections الأخرى:
- **الأجناس** (genres) - Pills ملونة
- **الكلمات المفتاحية** (keywords) - Pills بسيطة
- **القصة** (overview_ar / overview)
- **الممثلين** (cast) - صور دائرية مع الأسماء
- **المواسم والحلقات** (seasons/episodes) - للمسلسلات فقط
- **أعمال مشابهة** (similar)

### ❌ بيانات لا تُعرض:
- **الشعبية** (popularity) - تم إلغاؤها
- **Category** - فاضي في قاعدة البيانات

## 🗂️ الفرق بين الحقول

### النوع vs Type:
- **النوع (media_type)**: فيلم أو مسلسل
- **Type**: نوع المسلسل (Scripted, Documentary, Talk Show, Reality, etc)
- **Category**: غير مستخدم (null)

### العناوين:
- **title / name**: العنوان الرئيسي
- **title_ar / name_ar**: العنوان بالعربية
- **title_en / name_en**: العنوان بالإنجليزية
- **title_original / name_original**: العنوان الأصلي

### التواريخ:
- **release_date**: تاريخ إصدار الفيلم
- **first_air_date**: تاريخ أول حلقة (مسلسل)
- **last_air_date**: تاريخ آخر حلقة (مسلسل)

## 🎨 قواعد التصميم

1. **Mobile-First**: التصميم يجب أن يعمل على الموبايل والتابلت
2. **Desktop**: البوستر يسار + المعلومات يمين
3. **Responsive**: كل العناصر تتجاوب مع حجم الشاشة
4. **No Sandbox**: لا تستخدم sandbox في iframe
5. **No referrerPolicy**: لا تستخدم referrerPolicy في iframe
6. **Simple iframe**: فقط src, className, allowFullScreen, allow
